# Project Overview

## Objective

Deploy a self-hosted Nextcloud server using Ubuntu Linux.

## Goals

- Install Nginx
- Configure MariaDB
- Configure PHP
- Deploy Nextcloud
- Configure hostname
- Test connectivity
- Troubleshoot deployment issues
